# Courage the Cowardly Dog — Windows-Inspired Portfolio

Welcome to my personal portfolio site!  

This site is inspired by the iconic computer from Courage the Cowardly Dog, reimagined with a sleek, modern Windows 11 aesthetic. It blends nostalgia with a futuristic feel, giving visitors an interactive and playful way to explore my work.

## Features

- **Interactive "start menu"** interface styled after Windows 11
- Pinned "apps" showcasing projects and creative pages
- Recommended projects section with direct GitHub links
- Pop-up modal to display certifications
- Custom taskbar and user profile section for easy navigation

## Tech stack

- **React**: Handles the dynamic, component-based user interface and interactive behavior.
- **JavaScript**: Powers the logic, interactions, and dynamic content rendering across the site.
- **CSS**: Provides the styling, animations, and layout, recreating the Windows 11 feel.

## Usage

- Click any app icon to view projects, fun mock pages, or links.
- Check out recommended GitHub projects for recent highlights.
- Click user profile icons to view resume, about information, and certifications.
