# 💻 Courage the Cowardly Dog — Computer Portfolio Site

A fun and interactive "start menu"-style personal portfolio website inspired by Courage the Cowardly Dog and styled to look like a modern Windows 11 desktop.  

[🔗 **Live Demo**](https://cyristal-gems.github.io)

---

## 🧰 Built With

- **React** — framework powering all interactivity and state management.
- **Tailwind CSS** — for styling, layout, and responsiveness.
- **Framer Motion** — for animations and smooth transitions.
- **GitHub Pages** — for seamless deployment and hosting.

---

## 🚀 Features

- **Full-screen animated loading screen** with custom audio greeting.
- **Static pinned icons** that simulate app shortcuts.
- **"Muriel’s Finds"** section linking to personal GitHub projects.
- **Taskbar** styled to match Windows 11 aesthetic.
- **Certifications pop-up** showcasing professional credentials.
- Custom footer line with links to About, Resume, Certifications, and GitHub Repos.
